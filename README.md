# Emily & Ryder · 英语启蒙每日情景对话

这是一个专为幼儿设计的英语启蒙 Web 应用，旨在通过生活化的情景对话，帮助孩子在日常互动中自然地习得英语。

## 核心理念
*   **生活化：** 拒绝死记硬背，将词汇融入到吃饭、穿衣、出门、洗澡等真实生活场景中。
*   **不重复：** 提供 90 天全量不重复的学习计划，每天一个新场景，保持孩子的新鲜感。
*   **亲子交互：** 每日学习包含简单的对话和行动指南，方便父母与孩子高效互动。

## 功能架构
*   **90 天学习序列：** 系统根据起始日期自动排期，确保每天的任务都是唯一的，覆盖晨起、户外探索、情感交流等全方位启蒙。
*   **词汇系统：** 每个场景包含核心词汇及音标 (IPA) 标注，方便家长准确发音示范。
*   **权威资源：** 每个学习任务都配套了精选的 Bilibili 优质启蒙视频/儿歌链接，作为辅助学习资源。

## 目录结构
*   `curriculum.js`: 核心词库与场景定义，包含 90 天的学习路径。
*   `data.js`: 90 天自动化学习计划生成器。
*   `app.js`: 应用渲染与交互逻辑（侧边栏、每日任务、打卡功能）。
*   `style.css`: 现代化的玻璃拟态 UI 样式。
*   `index.html`: 主页面结构。

## 如何使用
1.  在 `data.js` 中设置 `startDate` 为开始学习的日期。
2.  打开应用，即可根据日期自动进入当天的学习场景。
3.  阅读对话、音标，跟随 Tip 提示与孩子互动。
4.  完成每日跟读与应用练习后进行打卡，查看学习连胜进度。
