# 英语启蒙卡片设计规格

## 概述
将现有黑板风格改为卡通可爱的场景化卡片界面，每个场景有对应卡通元素和柔和渐变背景。

## UI 设计

### 整体风格
- 圆角卡片设计 (border-radius: 24px)
- 柔和渐变背景，按场景区分颜色
- 可爱圆润的儿童风格

### 场景配色方案

| 场景 | 背景渐变 | 卡通元素 |
|------|----------|----------|
| 早上见面 | 淡粉色 #fff5f5 → #ffe4e4 | ☀️ 太阳 + 笑脸 |
| 出门 | 淡绿色 #f0fff4 → #d4f4dd | 👟 鞋子 + 🧥 外套 |
| 吃饭 | 淡黄色 #fffef5 → #fff8dc | 🍚 米饭碗 + 勺子 |
| 谢谢 | 淡紫色 #f5f0ff → #e8e0ff | 💜 爱心 |
| 道歉 | 淡蓝色 #f0f8ff → #e0f0ff | 🌈 彩虹 |
| 赞美 | 淡金色 #fffbf0 → #fff5d6 | ⭐ 星星 |
| 晚安 | 淡紫蓝 #f5f0ff → #e8e0f0 | 🌙 月亮 + ✨ 星星 |

### 组件设计

#### 1. 顶部卡通区
- 大号卡通图标 (64px)
- 居中显示
- 轻微弹跳动画

#### 2. 场景标题
- 圆体中文字体
- 场景名称 + 小图标
- 底部装饰线条

#### 3. 对话气泡
- 圆角气泡框 (border-radius: 20px)
- 英文句子大号字体 (24px)
- 中文解释小号字体 (16px)
- 左右交替排列（家长/孩子）

#### 4. 底部打卡区
- 勾选框样式
- 完成后动画

## 交互设计

### 动画效果
- 页面加载：卡片从下滑入 (0.5s ease-out)
- 气泡依次出现 (stagger 0.2s)
- 卡通元素轻微上下浮动 (infinite 3s)
- 打卡成功：勾选放大 + 绿色填充

### 响应式
- 移动端优先设计
- 最大宽度 480px
- 内边距自适应

## 实现计划

1. 创建新 HTML 结构（卡片式）
2. 定义场景配色 CSS 变量
3. 实现卡通图标组件
4. 实现对话气泡组件
5. 添加动画效果
6. 保持 data.js 数据兼容
